**User Requirements (UR)**

UR-1.
Клиент должен иметь возможность оставить отзыв по продукту с оценкой и комментарием.

UR-2.
Менеджер продукта должен иметь возможность просматривать отзывы по продуктам и периодам.

UR-3.
Аналитик должен иметь возможность анализировать feedback по динамике и уровню удовлетворённости.
